A small wrapper around last.fm in early development.

## Installation

The package can be installed by adding `last_fm_wrapper` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:last_fm_wrapper, "~> 0.1.0-alpha"}
  ]
end
```
